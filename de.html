TECH |

Webentwicklung & Data Science

> HTML / CSS

HTML (HyperText Markup Language) ist eine Auszeichnungssprache, die die Struktur und Inhalte von Webseiten definiert und mit speziellen Tags arbeitet, um Texte, Bilder und Tabellen zu strukturieren und darzustellen.

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die verwendet wird, um das Aussehen und Layout von HTML- und XML-Dokumenten zu definieren, indem sie Anweisungen für die Darstellung von Elementen auf verschiedenen Medien gibt.

> JavaScript

JavaScript ist eine vielseitige Programmiersprache, die hauptsächlich zur Erstellung interaktiver und dynamischer Webseiten verwendet wird, sowohl auf der Client- als auch auf der Serverseite.

> Python

Python ist eine universelle, plattformübergreifende Open-Source-Programmiersprache, die für ihre einfache Syntax, Vielseitigkeit und breite Anwendbarkeit bekannt ist.

||| Blender

Blender ist eine freie, mit der GPL lizenzierte 3D-Grafiksoftware, mit der sich modellieren, texturieren und animieren lässt.
